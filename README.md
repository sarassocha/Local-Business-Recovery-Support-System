## Project Description

This project focuses on the development of a **Local Business Recovery Support System**, a software application designed to support small businesses in their recovery process after the COVID-19 pandemic.

The system aims to improve **customer engagement, business visibility, and operational decision-making** by providing tools for monitoring customer activity, managing information, and generating analytical insights that help businesses optimize their operations.

This project is developed as part of the **Systems Analysis and Design course**, where the full systems engineering lifecycle is applied, including **system analysis, system design, project management, simulation, and prototype implementation**.

The final objective is to create a **functional prototype** that demonstrates how technology can help local businesses recover and improve their performance through data-driven decision making.


<br>

## Objectives

### General Objetive
Develop a prototype application that supports **local businesses in their post-pandemic recovery**, improving customer engagement and providing analytical tools for better decision-making.


### Specific Objectives

- Analyze the current challenges faced by local businesses after the pandemic.
- Design a system architecture that supports business management and customer interaction.
- Develop a functional prototype that demonstrates the main components of the proposed system.
- Implement tools for **customer engagement and basic analytics** to help businesses understand their performance.
- Evaluate the effectiveness of the system through testing and stakeholder feedback.


<br>

## Project Scope

The project focuses on the **design and prototype implementation of a support system for local businesses**.

The system will include features that help businesses:

- Manage business information and activities.
- Improve customer interaction and engagement.
- Analyze operational data to support decision-making.
- Identify opportunities for improving business performance.

The prototype will demonstrate the **core functionality and architecture of the system**, showing how technology can help optimize business operations and improve recovery after the pandemic.

This project integrates all phases of the **systems engineering process**, including:

- Systems Analysis
- Systems Design
- Project Management Planning
- System Simulation and Validation
- Prototype Implementation


<br>

## System Features (Prototype)

The proposed system prototype may include functionalities such as:

- Business information management
- Customer interaction tracking
- Basic analytics and reporting
- Data visualization for business insights
- Tools that help businesses understand customer behavior and improve engagement
